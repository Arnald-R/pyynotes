```python
class AgeError(Exception):
    pass
```


```python
def verify_age(age):
    try:
        if age < 0:
            raise AgeError("Age cannot be Negative")
        print(f"Age verified:{age}")
    except AgeError as e:
        print(f"Error:{e}")
```
