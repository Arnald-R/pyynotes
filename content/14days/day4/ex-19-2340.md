```python
from functools import partial
```


```python
def multiply(x, y):
    return x * y
```


```python
double = partial(multiply, y=2)
```


```python
triple = partial(multiply, y=3)
```


```python
print(f"Double 5:{double(5)}")
```

    Double 5:10



```python
print(f"triple 5:{triple(5)}")
```

    triple 5:15
