     === Decorator Example ===



```python
def timer_decorator(func):
    def wrapper(*args, **kwargs):
        start = datetime.now()
        result = func(*args, **kwargs)
        end = datetime.now()
        print(f"function took:{end-start} time for the exection")
        return result
    return wrapper
```


```python
@timer_decorator
def slow_function():
    return sum(i * i for i in range (1000000))
```
