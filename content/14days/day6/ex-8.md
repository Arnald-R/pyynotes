```python
class Mathoperations:
    @staticmethod
    def add(x, y):
        return x + y

    def multiply(x, y):
        return x * y
```


```python
print(f"add :{Mathoperations.add(5, 3)}")
```

    add :8



```python
print(f"multiply :{Mathoperations.multiply(5, 4)}")
```

    multiply :20
