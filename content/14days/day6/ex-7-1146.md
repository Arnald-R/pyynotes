```python
from abc import ABC, abstractmethod
```


```python
class Vehicle(ABC):
    @abstractmethod
    def start(self):
        pass

    
```


```python
class Car(Vehicle):
    def start(self):
        return "engine started"
```


```python
car = Car()
```


```python
print(car.start())
```
