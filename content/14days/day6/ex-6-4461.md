```python
class Shape:
    def area(self):
        return 0 
```


```python
class Square(Shape):
    def __init__(self, side):
        self.side = side

    def area(self):
        return self.side ** 2
```


```python
square = Square(5)
```


```python
print(f"Square area:{square.area()}")
```
