```python
#Coroutine for producing  Fibanocci Numbers
```


```python
def fibonacci_coroutine():
    a, b = 0 ,1
    while True:
        yield a
        a, b = a, a + b
```


```python
coro = fibonacci_coroutine()

```


```python
for _ in range(6):
    print(next(coro), end =" ")
```
