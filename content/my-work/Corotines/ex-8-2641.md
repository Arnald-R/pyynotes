```python
#Coroutine for Logging messages
```


```python
def logger():
    print("logger started")
    while True:
        message = yield 
        print(f"log : {message}")
```


```python
coro = logger()
```


```python
next(coro)
```

    logger started



```python
coro.send("process started")
```

    log : process started



```python
coro.send("process completed")
```

    log : process completed
