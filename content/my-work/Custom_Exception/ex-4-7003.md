```python
class CustomFileError(Exception):
    pass
```


```python
def read_file(file_path):
    try:
        with open(file_path,'r') as file:
            return file.read()
    except FileNotFoundError as e:
        raise CustomFileError(f"File {file_path} not found") from e
```
