```python
#Default values in the Dataclasses
```


```python
from dataclasses import dataclass
```


```python
@dataclass
class Book:
    Book_name : str
    Author    : str
    price     : float
```


```python
book = Book(Book_name = "logic first",Author = "Dennis", price = 99.9)
```


```python
print(book)
```
