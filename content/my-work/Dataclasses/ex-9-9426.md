```python
#Inheriting from dataclass
```


```python
from dataclasses import dataclass
```


```python
@dataclass
class Animal:
    name : str
```


```python
@dataclass
class Dog(Animal):
    breed : str
```


```python
dog = Dog(name ="greeda", breed ="puppy")
```


```python
print(dog)
```
