```python
from unittest.mock import patch
```


```python
class MyClass:
    def method(self):
        return "Original method"
```


```python
with patch("__main__.MyClass") as MockClass:
    instance = MockClass.return_value
    instance.method.return_value = "Mocked_Method"
    print(instance.method())
```
