```python
from typing import Any
```


```python
def stringify(value :Any) -> str:
    return str(value)
```


```python
print(stringify(2.0))
```

    2.0



```python
print(stringify(2))
```

    2



```python
print(stringify([1,23,4]))
```

    [1, 23, 4]
