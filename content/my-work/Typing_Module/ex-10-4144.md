```python
from typing import TypedDict
```


```python
class User(TypedDict):
    name = str
    age  = int
    email= str
```


```python
def display(user : User) -> str :
    return f"{user['name']}({user['age']}) and be reached {user['email']}"
```


```python
user = {"name": "Arnald","age": 23,"email": "arnald@gmail.com"}
```


