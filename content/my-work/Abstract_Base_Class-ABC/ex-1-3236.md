```python
#Basic Abstract class
```


```python
from abc import ABC, abstractmethod
```


```python
class Animal(ABC):
    @abstractmethod
    def make_sound(self):
        pass
        
```


```python
class Dog(Animal):
    def make_sound(self):
        return "woof"
```


```python
dog = Dog()
```


```python
print(dog.make_sound())
```
