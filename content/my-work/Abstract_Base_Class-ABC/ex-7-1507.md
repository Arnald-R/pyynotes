```python
#Connecting Abstract Base class with interfaces
```


```python
from abc import ABC, abstractmethod
```


```python
class Flyable(ABC):
    @abstractmethod
    def fly(self):
        pass
```


```python
class Bird(Flyable):
    def fly(self):
        print("Bird is flying")
```


```python
bird = Bird()
```


```python
bird.fly()
```

    Bird is flying
