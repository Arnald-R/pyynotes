```python
#Abstract Base Class with class methods 
```


```python
from abc import ABC,abstractmethod
```


```python
class Database(ABC):
    @classmethod
    @abstractmethod
    def connect(cls):
        pass
```


```python
class MysqlDatabase(Database):
    @classmethod
    def connect(cls):
        print("connecting to Mysql")
        
```


```python
MysqlDatabase.connect()
```
