```python
#Original Function 
def greet(name):
    return f"Hello, {name}"
```


```python
#Monkey Patching Function
def new_greet(name):
    return f"Hi, {name}"
```


```python
greet = new_greet #Replace the original function with the new one
```


```python
#Test the patched function
```


```python
print(greet("Arnald"))
```
