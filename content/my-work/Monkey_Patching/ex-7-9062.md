```python
#Original function
def divide(a,b):
    return a / b
```


```python
#Monkey Patch to handle division by zero
def safe_divide(a,b):
    if b == 0:
        return "Error : Division by zero"
    return a / b
```


```python
divide = safe_divide
```


```python
print(divide(10,0))
```

    Error : Division by zero



```python
print(divide(10,2))
```
