```python
#Monkey Patching for Debugging
```


```python
#Original fuction
def add(a,b):
    return a + b
```


```python
#Monkey patch to add debugging
def debug_add(a,b):
    print(f"Adding {a} and {b}")
    return a + b
```


```python
add = debug_add #Replace the add function
```


```python
#Test the add function
print(add(2,3))
```
