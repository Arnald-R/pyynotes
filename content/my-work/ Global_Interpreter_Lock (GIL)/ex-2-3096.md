```python
#MultiThreading with the GIL (I/O - bound)
```


```python
import threading
import time
```


```python
#I/O- bound task
def io_bound_task():
    time.sleep(2) #Simulate I/O operation
    print("I/O Task done")
```


```python
#Start two threads
threads = []
for _ in range(2):
    thread = threading.Thread(target=io_bound_task)
    threads.append(thread)
    thread.start()
```

    I/O Task done



```python
for thread in threads:
    thread.join()
```
